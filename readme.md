## Purpose

Autonomous AI systems need more than ethics statements and documentation.  
They need **enforceable rules**.

This repository provides:
- Machine-readable governance policies
- Decision boundaries for AI execution
- Human-first constraints on autonomy
- Inputs for certification and audit

These policies are intended to sit **above execution engines** such as:
- Agent frameworks (e.g. Moltbot)
- Automation platforms (e.g. UiPath)
- Orchestrators and schedulers

**Agents execute.  
Metronisys governs.**
